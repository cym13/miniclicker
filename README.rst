Description
===========

Mini autoclicker

X11 only, took some code from xwarppointer

Documentation
=============

::
    Usage: ./miniclicker NUM [DEL]
    Clicks where the pointer is at script launch

    Args:
        NUM   Number of clicks, infinite if 0
        DEL   Delay between clicks (mlls)
